釣行記

加賀フィッシングエリア（トラウト・管釣り）【2007年2月28日 * Wed】

今日は友人に誘われ、加賀フィッシングエリアさんにお邪魔しました。

チョット頑張りました！
到着は８時！珍しく朝から釣りしてます＾＾
稲村は半夜型人間なので、朝にはめっぽう弱いです。。。
この日も、約2ヶ月振りに朝日を拝みました。

だいぶ春らしいポカポカ陽気になり、日中はアウターを着ていると暑いぐらいでした。

今年初タンポポ発見！！






そんなこんなで釣り開始！！
まずはお気に入り、ノリーズ・クランキンピューパをキャスト。


マジでこのルアー釣れます！！
加賀Ｆ.Ａみたいに魚が表層にいる場合は特に強いです。
使用頻度はダントツ№１☆

別にワタクシ、マルキューさんの回し者ではありませんがこのルアーはゴリ押しデス！！
簡単なコツがありますので、気になった方はお声がけお願いいたしまーす☆


その後も色を変えながら３０匹ぐらいキャッチ。
次はさらに沖にいる魚を狙います。

ピュア5ｇと鱒玄人３.７ｇＬＤをローテーションさせていると、鱒玄人にヒット！


沖目にいる魚は活性が高くアタリもでかくて楽しい☆
やはりセオリー通り、『遠投・表層』がキモみたいです。

午後になりライズが始まると、フライの方が爆釣してました！！
今度はフライタックル持っていこう。。。


午後になると若干渋くなりましたが、それでも１日中楽しめました♪
広いので、のんびり釣るには最適です。
